# Planning

- Cryptographic hash: keccak256;
- Digital signature: Ed25519;
- Immutable ledger: merkle trie (LevelDB as the underlying storage);
- P2P network: with libp2p;
- Consensus algorithm: Proof of Work (PoW);
- Block validation: Mining;
